# Training Rules

训练目标优先级：增肌塑形优先，同时保留功能性维护。

每周训练覆盖原则：频率不固定时，优先覆盖大肌群和当前最需要推进的训练目标，不假设用户明天一定训练。

增肌 / 塑形 / 功能性维护原则：主训练保留渐进刺激，辅助训练补足弱项，核心和活动度用于维持动作质量。

强度管理原则：主训练通常 RPE 7-8；状态好时可到 RPE 8-9；疲劳高或动作质量下降时降重、减组或延长休息。

疲劳处理原则：不能用户一说累就直接停止训练；先判断全身疲劳、局部疲劳、异常不适和当前完成度。

疼痛处理原则：尖锐痛、放射痛、麻木、胸闷、头晕、关节不稳定时停止相关动作并给安全替代，不做医疗诊断。

器械替换原则：优先保持训练目标和动作模式，再根据可用器械替换。

不固定频率下的训练安排原则：每次训练要有完整价值，优先安排当天最值得完成的主训练。

场地适配原则：根据空间、器械、人流观察和安全限制调整动作。

保守策略限制：不能用户一说累就直接停止训练。
